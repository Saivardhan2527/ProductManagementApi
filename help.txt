How to Run the Project
Step 1: Install Required Software

Make sure the following software is installed on your computer:

Java JDK 17 or later (Java 23 recommended)
Apache Maven 3.9+
Git
Postman (for testing APIs)

Verify the installations:

java -version
mvn -version

Both commands should display the installed versions.

Step 2: Clone the Repository

Open Command Prompt or PowerShell.

Run:

git clone https://github.com/Saivardhan2527/ProductManagementApi.git

Move into the project folder:

cd ProductManagementApi
Step 3: Configure the Database

This project uses MongoDB.

Open:

src/main/resources/application.properties

Update the MongoDB configuration with your own database details.

Example:

spring.data.mongodb.host=YOUR_HOST
spring.data.mongodb.port=YOUR_PORT
spring.data.mongodb.database=YOUR_DATABASE
spring.data.mongodb.username=YOUR_USERNAME
spring.data.mongodb.password=YOUR_PASSWORD

Note: If you're using a cloud MongoDB instance or another provider, replace these values with your own credentials.

Step 4: Download Project Dependencies

Run:

mvn clean install

This command:

Downloads all required libraries
Compiles the Java code
Runs the tests
Creates the target folder

Wait until you see:

BUILD SUCCESS
Step 5: Start the Application

Run:

mvn spring-boot:run

When the application starts successfully, you'll see:

Started Main
Tomcat started on port(s): 8080
Step 6: Open the Application

Open your browser:

http://localhost:8080

The demo HTML page will appear.

Note: The main functionality of this project is provided through REST APIs. The HTML page is only a simple demo.
